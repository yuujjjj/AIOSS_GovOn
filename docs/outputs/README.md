# 산출물 관리 (Outputs Management)

프로젝트 마일스톤별 산출물을 체계적으로 관리합니다.

## 폴더 구조

```
outputs/
├── M1_Planning/       # 기획 및 설계 (Week 1-4)
├── M2_MVP/            # 핵심 기능 구현 (Week 5-8)
├── M3_Optimization/   # 고도화 및 최적화 (Week 9-12)
└── M4_Testing/        # 테스트 및 문서화 (Week 13-16)
```
